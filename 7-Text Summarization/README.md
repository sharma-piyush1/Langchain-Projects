# Text Summarization

Small Streamlit demo project that converts long text into concise summaries using a language model and LangChain utilities.

## Contents
- app.py — main Streamlit application (text input / file upload + summarization pipeline)
- requirements.txt — Python dependencies (create if missing)
- .env — environment variables (do not commit)
- README.md — this file

## Features
- Paste text or upload documents to summarize
- Option to choose model, temperature, and summary length
- Simple UI with Streamlit

## Prerequisites
- Windows
- Python 3.9+
- pip

## Quick setup (PowerShell)
1. Create & activate a virtual environment:
```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

2. Install dependencies:
